A social network where autistic people can connect, share, and find support

Features

Edit Profile


Users can edit their profile information, including name, bio, and preferences.

Read Articles


Users can read articles and access full content by clicking “Read more”.

Create Posts


Users can create posts with a title, theme, and message to share with the community.

About Autistic Eye


This project was inspired by my desire to support autistic individuals. I have witnessed the challenges my brother faces and wanted to create a safe, supportive online community.

Part of my portfolio for Holberton School.
